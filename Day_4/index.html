Spread Operator

Spread operator expands the array and stream and gives the access of individual elements

Ternary Operator

It is a shorthand property for an if-else statement. It returns one or two values based on a condition.

First class function

In first class function, we can pass a function to a variable and we treat the variable as a function.

We can pass a function as an argument inside another function

It can return a function from a function
